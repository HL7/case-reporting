    <div xmlns="http://www.w3.org/1999/xhtml"><h2>Reportable Condition Trigger Codes (RCTC)</h2><div><p>This is the draft trigger codes table. This list of codes is intended use is for filtering encounters in healthcare settings to screen for public health reportable conditions. It is intended to be used in the provider EHR system environment as lookup table to identify cases that should be forwarded to public healthas an intitial case report. The standard codes in this list are directed at diagnosis, laboratory results and laboratory order for reportable conditions. Specifically it consist if ICD-10CM, SNOMED CT and LOINC codes. It may be expanded to include other code systems in future.</p>
