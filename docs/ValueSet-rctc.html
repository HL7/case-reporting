Name:Reportable Condition Trigger Codes (RCTC)
Definition:

This is the draft trigger codes table. This list of codes is intended use is for filtering encounters in healthcare settings to screen for public health reportable conditions. It is intended to be used in the provider EHR system environment as lookup table to identify cases that should be forwarded to public healthas an intitial case report. The standard codes in this list are directed at diagnosis, laboratory results and laboratory order fro reportable conditions. Specifically it consist if ICD-10CM, SNOMED CT and LOINC codes. It may be expanded to include other code systems in future. It is derived from:1) The RKCMS tables list of code of which currently only for Chlamydia and Pertussis and Lead exist. 2) edited RCMT tables excluding out of scope and generic conditions 3) Cecil Lynch's Public Health Ontology. 4) CSTE's Technical Implementation Guides (TIG) documents. 5) ICD-10 Master Mapping Reference Table from ISDS. These codes are are deduplicated and sorted by code system. The list of conditions it encompasses is covered in the tab 'ConditionsCoveredbyCodeSystem' Tab [todo url for this].
