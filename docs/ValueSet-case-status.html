CodeDisplayDefinition
in-progressIn ProgressThis is a partial (e.g. initial, interim or preliminary) case. Data may be incomplete or unverified.
suspendedSuspendedIncomplete or unavailable because the case has been temporarily halted.
abortedAbortedIncomplete or unavailable because the case was abandoned prior to completion.
completedCompletedThe case is complete
reopenedReopenedSubsequent to being completed, the case has been reopened. This includes any change in the results, diagnosis, narrative text, or other content that has been issued.
entered-in-errorEntered in ErrorThis electronic record of this event should never have existed, though it is possible that real-world decisions were based on it. (If real-world activity has occurred, the status should be "cancelled" rather than "entered-in-error".)
unknownUnknownThe authoring system does not know which of the status values currently applies. Note: This concept is not to be used for "other" - one of the listed statuses is presumed to apply, it's just not known which one.
